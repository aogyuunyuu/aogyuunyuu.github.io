テキストランダム抽出器

左側の入力欄に入力されたリストから空白行を削除して「抽出する行数」分のリストが右の欄に出力されます。
「コピーする」ボタンを押下すると右の欄に出力された結果をコピーできます。
indexに戻る
import random def extract_random_lines(event): input_text = Element("inputText").value lines = input_text.splitlines() # 改行で分割 # 空白行を無視して重複を排除 unique_lines = list(set(line for line in lines if line.strip())) # 抽出する行数を取得 try: line_count = int(Element("lineCount").value) except ValueError: line_count = 20 # 行数が範囲外の場合は調整 line_count = min(line_count, len(unique_lines)) random.shuffle(unique_lines) # ランダムにシャッフル output_lines = unique_lines[:line_count] # 指定された数の行を抽出
