テキストランダム抽出くん

左側の入力欄に入力されたリストから重複行と空白行が削除され「抽出する行数」分のリストが右の欄にランダムで出力されます。
テキストファイルを入力欄に直接ドロップすることもできます。
「コピーする」ボタンを押下すると右の欄に出力された結果をコピーできます。
    入力された行数: 0 行

    import random def extract_random_lines(): input_text = Element("inputText").value lines = input_text.splitlines() unique_lines = list(set(line for line in lines if line.strip())) line_count = int(Element("lineCount").value) if Element("lineCount").value.isdigit() else 20 line_count = min(line_count, len(unique_lines)) output_lines = random.sample(unique_lines, line_count) return output_lines
